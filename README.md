# iyf-s10-week-05-kimderlywamboi8# 🚀 Pro Task Manager App

A modern, interactive **To-Do List Web Application** built with HTML, CSS, and JavaScript.
This app allows users to manage tasks efficiently with features like filtering, editing, due dates, and drag-and-drop reordering.

---

## ✨ Features

* ✅ Add new tasks
* ✏️ Edit tasks (double-click)
* 📅 Set due dates
* 🔄 Drag & drop to reorder tasks
* ✔️ Mark tasks as completed or active
* ❌ Delete tasks
* 🔎 Filter tasks (All / Active / Completed)
* 💾 Save tasks using Local Storage (data persists after refresh)
* 🔔 Notifications for due tasks
* 📊 Task counter

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling (modern UI design)
* **JavaScript (Vanilla JS)** – Functionality & interactivity
* **Local Storage API** – Data persistence

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder in **VS Code**
3. Open `index.html` in your browser
   OR
   Use **Live Server** extension for better experience

---

## 🎯 How to Use

* Type a task and optionally select a due date
* Click **Add** to create a task
* Click on a task to mark it as completed
* Double-click a task to edit it
* Use filter buttons to view:

  * All tasks
  * Active tasks
  * Completed tasks
* Drag tasks to reorder them
* Tasks are automatically saved in your browser
